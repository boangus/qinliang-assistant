# 沁凉助手 - 潮热盗汗 AI 智能管理工具

## 项目简介

沁凉助手是一款专为 ADT（雄激素剥夺治疗）相关潮热患者设计的智能管理工具。基于 Vue 3 + Tailwind CSS + ECharts 构建，采用单文件 HTML 架构，无需任何构建工具即可运行。

## 核心功能

- **智能症状记录** - 记录潮热发作的时间、严重程度、诱因、伴随症状和干预措施
- **AI 风险预测** - 基于历史数据的 24 小时风险预测与个性化建议
- **数据统计分析** - 趋势图、饼图、热力图、诱因分析、干预效果对比
- **耳穴压豆指导** - 10 个穴位的定位、手法指导和 30 秒按压计时器
- **中医体质评估** - 9 种体质类型 × 8 题 = 72 题标准化评估，含雷达图结果展示
- **睡眠质量评估** - PSQI 匹兹堡睡眠质量指数量表（12 题，7 维度）
- **VMS 研究问卷** - 9 步向导式问卷，含 HFNS、HFRDIS、HFBBS-Men 等量表
- **分级治疗推荐** - 三级治疗路径（生活方式→中医特色→药物参考）
- **健康科普知识库** - 潮热机制、诱因管理、中医调理、西医药物、生活技巧
- **潮热盗汗工具箱** - 首页醒目入口，一键直达饮食工具、CBT工具、耳穴指导
- **饮食管理工具** - 饮食红绿灯、每日饮食打卡、饮水提醒、案例展示
- **CBT 认知行为工具** - 4-7-8呼吸法、腹式呼吸、渐进式肌肉放松、认知重构练习、睡眠卫生检查表、分层降温技术
- **CBT 7 天入门计划** - 结构化每日练习，7 天掌握应对潮热的核心 CBT 技能
- **运动处方** - 有氧/抗阻运动记录，周目标跟踪，近 4 周趋势可视化
- **每日 30 秒检查** - 睡眠、情绪、疲劳、运动、饮食、饮水的快速记录与趋势分析
- **健康报告导出** - 基于近 30 天数据生成可打印的 PDF 风格报告
- **个人中心** - 用户信息编辑、数据导出（JSON）、数据清除

## 技术栈

- **Vue 3** (CDN) - 响应式 UI 框架
- **Tailwind CSS** (CDN) - 原子化 CSS 样式
- **ECharts 5** (CDN) - 数据可视化
- **Marked.js** (CDN) - Markdown 渲染

## 使用方式

直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何依赖。

```bash
# 方式一：直接打开
open index.html

# 方式二：使用本地服务器
npx serve .
```

## 数据存储

所有数据存储在浏览器的 localStorage 中，包括：
- 症状记录数据
- 用户个人信息
- 体质评估结果
- 量表评估结果
- 每日检查记录
- 饮食打卡与饮水记录
- 运动记录
- CBT 练习记录与 7 天计划进度

支持通过个人中心页面导出所有数据为 JSON 文件。
